📖 MacArkPet Web 管理面板 — 使用指南

      
访问地址：http://localhost:19191 · 随 MacArkPet 自动启动

    


    

      
🚀 快速上手

      
    
        
  1. 启动桌宠 — 在原生启动器中选择角色并点击「Launch Full Character」
    2. 
        
  2. 配置触发规则 (🎯 标签) — 设置哪些 App/网站触发什么情境
    3. 
        
  3. 管理角色 (📋 标签) — 查看/编辑角色设定，或从 prts.wiki 自动生成
    4. 
        
  4. 实时监控 (👁️ 标签) — 查看桌宠"看到"的屏幕内容和触发的台词
    5. 
        
  5. 远程控制 (🖥️ 标签) — 给桌宠发送互动命令、调整设置
    6. 
      

    


    

      
🎯 触发规则管理

      
每个规则是一个「情境分类 → 关键词列表」的映射。当桌宠检测到前台窗口标题或 Bundle ID
      包含某个关键词时，会触发对应情境的角色台词。

      
操作说明

      
    
        
  • 添加关键词 — 在分类底部的输入框中输入关键词，按 Enter
    • 
        
  • 删除关键词 — 点击关键词标签右上角的 ×
    • 
        
  • 添加分类 — 点击「+ 添加分类」，输入分类名称
    • 
        
  • 批量操作 — 支持清空分类、跨分类复制、TOML 导入/导出
    • 
      

      
配置文件位置：~/Library/Application Support/MacArkPet/Config/ScreenTriggers.toml

    


    

      
📋 角色管理

      
角色列表

      
    
        
  • 按阵营/职业/种族筛选，或搜索角色名/ID
    • 
        
  • 点击展开查看详细设定（性格、语言风格、背景故事、台词）
    • 
        
  • 点击 ✏️ 编辑角色设定 修改角色信息
    • 
        
  • 点击 🎭 编辑台词 进入台词编辑器
    • 
        
  • 点击 🗑 删除 彻底删除角色（含台词数据，不可撤销）
    • 
      

      
台词编辑器

      
    
        
  • 每个情境可有多组台词，每组支持好感度门槛和冷却时间
    • 
        
  • affection_* 前缀的情境需要好感度 ≥ 60 才触发
    • 
        
  • 从下拉菜单选择情境类型添加到角色
    • 
      

      
🤖 智能体生成

      
    
        
  • 输入角色名（中文），点击生成 → 自动从 prts.wiki 爬取数据
    • 
        
  • 开启 AI 润色 开关后，会用 DeepSeek 对内容进行润色
    • 
        
  • 生成结果可逐字段编辑，确认无误后保存到角色库
    • 
        
  • 首次使用需在 ⚙️ 中配置 AI API Key (该配置也会用于桌宠的日常总结、日记生成、节日彩蛋等陪伴功能)
    • 
      

    


    

      
👁️ 实时监控

      
    
        
  • 实时显示桌宠"眼中"的屏幕内容：前台 App、窗口标题、匹配的情境
    • 
        
  • 时间线展示最近 30 次活动记录
    • 
        
  • 触发历史持久化保存，可回溯过往匹配记录
    • 
        
  • 手动测试 — 在下方下拉选择情境 → 选择角色 → 点击触发，模拟屏幕变化
    •

全局设置

  • 🖱️ 点击穿透 — 开启后鼠标事件穿透桌宠窗口，不影响桌面操作
    • 
        
  • 📌 置顶 — 桌宠始终显示在所有窗口之上
    • 
      

      
远程控制命令

      
    
        
  • 👆 互动 / 🍽️ 喂食 — 提升好感度
    • 
        
  • 😴 休息 / 💤 睡觉 — 恢复体力
    • 
        
  • ✨ 特殊 — 触发角色的特殊动作

  • 📍 重置位置 — 将桌宠移回默认位置

启动新桌宠

      
从下拉菜单选择已安装的角色模型，点击「🚀 启动」即可在 WebUI 中直接启动新桌宠。

    


    

      
🔧 常见问题

      

        桌宠不动了怎么办？
        
尝试在 🖥️ 标签中点击「📍 重置位置」。如果不行，关闭桌宠后重新启动。

      

      

        如何让桌宠说更多话？
        
在 🎯 标签中为不同情境配置更多关键词。切换到不同的程序和页面，桌宠会根据情境变化自动触发台词。

      

      

        如何添加新角色？
        
两种方式：① 在 📋 标签的智能体区域输入角色名，从 prts.wiki 自动生成；
        ② 点击「➕ 新建角色」手动填写基本信息。

      

      

        AI 功能 (润色/日记/总结) 怎么配置？
        
在 📋 标签的智能体区域点击 ⚙️ 按钮，填入你的 DeepSeek / OpenAI API Key 和模型名称，点击保存。此配置全局生效，桌宠晚间总结、日记以及节假日对话均会使用该配置调用 AI。

      

      

        角色台词怎么修改？
        
在角色列表中展开角色详情，点击「🎭 编辑台词」，即可添加/删除/修改台词情境和内容。

      

      

        数据存在哪里？
        
所有数据存储在本地：

        ~/Library/Application Support/MacArkPet/ — 用户配置文件

        MacArkPet.app/Contents/Resources/ — 角色数据文件

      

    


    

      
📡 API 参考

      
WebUI 使用以下 API 端点。所有端点在 http://localhost:19191 提供服务。

      

        
端点方法说明

        
/api/statusGET服务器状态

        
/api/configGET/PUT触发规则配置

        
/api/config/exportGET导出 TOML

        
/api/config/importPOST导入 TOML

        
/api/profilesGET角色列表

        
/api/profiles/detailGET角色详情（含台词）

        
/api/profiles/savePOST保存角色设定

        
/api/profiles/deletePOST删除角色

        
/api/profiles/createPOST新建角色

        
/api/profiles/save-dialoguesPOST保存台词

        
/api/agent/generatePOST智能体生成角色

        
/api/agent/savePOST保存智能体结果

        
/api/contextGET当前屏幕上下文

        
/api/context/streamGETSSE 实时推送

        
/api/historyGET/POST触发历史

        
/api/petsGET桌宠列表

        
/api/pets/commandPOST桌宠命令

        
/api/pets/launchPOST启动桌宠

        
/api/pets/availableGET可启动角色

        
/api/settingsGET/PUT全局设置

        
/api/ai-configGET/PUTAI 配置

        
/api/auditGET角色数据审查
