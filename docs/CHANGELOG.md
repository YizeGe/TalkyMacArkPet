## 多桌宠管理（v0.1.0 → v0.1.2）

### 核心改动

- **单例 → 数组**：`PetWindowController` 从单个实例改为数组管理，支持同时运行 N 个桌宠
- **单 AP 图标**：所有桌宠共用一个菜单栏图标，通过二级子菜单分别控制
- **独立存档**：每个角色单独保存好感度/体力/签到数据

### 菜单结构

```
┌─ AP ───────────────────────┐
│ 🖥 打开启动器                │
│────────────────────────────│
│ 缪尔赛思 ▸                  │  ← 每个宠物的子菜单
│   ├ 开心一下 / 特殊动作       │
│   ├ 坐下休息 / 睡一会儿 / 喂食 │
│   ├ ──                      │
│   ├ 查看状态                 │
│   └ 关闭                    │
│ 澄闪 ▸                      │
│ 荒芜拉普兰德 ▸               │
│────────────────────────────│
│ ⭐ 全部操作                  │  ← 批量操作所有宠物
│────────────────────────────│
│ ☐ 点击穿透 / ☑ 保持置顶      │  ← 全局开关
│ ↺ 重置位置                   │
│────────────────────────────│
```

## Web 管理面板增强 + AI 润色（v0.1.2 → v0.2.0）

### 角色管理 CRUD

- **删除角色** 🗑 — 后端 API + 前端确认弹窗，同步清理 Profile/Dialogues/缓存
- **新建角色** ➕ — 弹窗表单输入 ID/名称/种族/职业/背景/性格等，一键保存
- **台词编辑器** 🎭 — 独立弹窗，支持：
  - 添加/删除台词情境（12+ 种预设）
  - 添加/删除台词组和单条台词
  - 编辑好感度门槛和冷却时间
- **编辑角色设定增强** — 新增种族、出身地、职业、生日、身高字段

### AI 智能体润色

- **LLM 润色引擎** — 爬虫扒完 prts.wiki 后自动调用 DeepSeek API
  - 润色性格描述、语言风格、对博士态度、背景故事
  - 生成符合角色性格的 12 种屏幕态度台词
  - 优化对话台词文本
- **AI 配置管理** — Web UI 内嵌配置面板
  - 支持 OpenAI 兼容 API（DeepSeek / OpenAI 等）
  - 配置持久化到 `~/.config/macarkpet/ai.json`
- **AI 开关** — 生成按钮旁的 toggle，可随时关闭 AI 润色

### 后端 API 新增

| 端点 | 方法 | 功能 |
|------|------|------|
| `/api/profiles/delete` | POST | 删除角色 |
| `/api/profiles/create` | POST | 新建角色 |
| `/api/profiles/save-dialogues` | POST | 保存台词 |
| `/api/ai-config` | GET/PUT | AI 配置读写 |
